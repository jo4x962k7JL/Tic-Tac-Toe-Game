# Tic-Tac-Toe-Game
Simple text-based implementation

## Example 1

```
$ python3 Tic-Tac-Toe.py

0 | 1 | 2              |   |
---------           -----------
3 | 4 | 5              |   |
---------           -----------
6 | 7 | 8              |   |

You are "x", Select a spot(0~9): 1

0 | 1 | 2            o | x |
---------           -----------
3 | 4 | 5              |   |
---------           -----------
6 | 7 | 8              |   |

You are "x", Select a spot(0~9): 4

0 | 1 | 2            o | x |
---------           -----------
3 | 4 | 5              | x |
---------           -----------
6 | 7 | 8              |   | o

You are "x", Select a spot(0~9): 7

0 | 1 | 2            o | x |
---------           -----------
3 | 4 | 5              | x | o
---------           -----------
6 | 7 | 8              | x | o

You Win!
```
